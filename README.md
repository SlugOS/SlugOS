# SlugOS
A minimal OS (32bit ARM fork) written in C and assembly.

## Supported hardware
| Device        | Supported     |
| ------------- | ------------- |
| [Integrator/CP](https://developer.arm.com/documentation/dui0159/b/introduction/about-the-integrator-cp)| yes |

## Why we support so few devices
Unlike x86 (what most computers use), ARM is a very fractured architecture; some ARM chips use entirely different instruction sets, and it is far less standardized compared to typical computers.

## Features
This is a very minimal and pretty much useless version of SlugOS.
